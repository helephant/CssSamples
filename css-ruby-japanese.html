Japanese text with furigana

Simple example:
新あたらしい

More complicated example:
私わたしは日本語にほんごを勉強べんきょうします。

Using the complex gem syntax (doesn't work in any browser):
私は日本語を勉強します。 わたしにほんごべんきょう
